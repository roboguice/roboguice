public abstract class RoboSplashActivity
extends android.app.Activity

An activity that can be used to display a splash page while initializing the guice injector in the background. Use of this class is definitely not required in order to use RoboGuice, but it can be useful if your app startup times are longer than desired. To use, simply override onCreate to call setContentView. Then override startNextActivity to specify where to go next.
